# иҺ·еҸ–е…ғзҙ зҡ„е®ҪеәҰе’Ңй«ҳеәҰ

# дҪҚзҪ®positionеұһжҖ§д»Ӣз»Қ вңЁ

  • staticпјҡй»ҳи®ӨеҖјгҖӮдҪҚзҪ®и®ҫзҪ®дёәstaticзҡ„е…ғзҙ пјҢжӯЈеёёж–ҮжЎЈжөҒзҡ„дҪҚзҪ®пјҢжӯӨж—¶ top, right, bottom, left е’Ң z-index еұһжҖ§ж— ж•ҲгҖӮ
  • inheritпјҢи§„е®ҡеә”иҜҘд»ҺзҲ¶е…ғзҙ з»§жүҝ position еұһжҖ§зҡ„еҖјгҖӮ
  • fixedпјҡз”ҹжҲҗеӣәеҜ№е®ҡдҪҚзҡ„е…ғзҙ гҖӮй»ҳи®Өжғ…еҶөдёӢпјҢеҸҜе®ҡдҪҚдәҺзӣёеҜ№дәҺеұҸе№•и§ҶеҸЈпјҲviewportпјүзҡ„дҪҚзҪ®гҖӮйҖҡиҝҮ вҖңleftвҖқ, вҖңtopвҖқ, вҖңrightвҖқ д»ҘеҸҠ вҖңbottomвҖқ еұһжҖ§иҝӣиЎҢи§„е®ҡгҖӮе…ғзҙ зҡ„дҪҚзҪ®еңЁеұҸе№•ж»ҡеҠЁж—¶дёҚдјҡж”№еҸҳгҖӮеҰӮпјҡеӣһеҲ°йЎ¶йғЁжҢүй’®гҖӮ
  • absoluteпјҡз”ҹжҲҗз»қеҜ№е®ҡдҪҚзҡ„е…ғзҙ гҖӮзӣёеҜ№дәҺи·қиҜҘе…ғзҙ жңҖиҝ‘зҡ„йқһ static е®ҡдҪҚзҘ–е…Ҳе…ғзҙ иҝӣиЎҢе®ҡдҪҚгҖӮжӯӨе…ғзҙ зҡ„дҪҚзҪ®еҸҜйҖҡиҝҮ вҖңleftвҖқгҖҒвҖқtopвҖқгҖҒвҖқrightвҖқ д»ҘеҸҠ вҖңbottomвҖқ еұһжҖ§жқҘи§„е®ҡгҖӮ
  • relativeпјҡз”ҹжҲҗзӣёеҜ№е®ҡдҪҚзҡ„е…ғзҙ пјҢзӣёеҜ№дәҺзӣёеҜ№дәҺжӯЈеёёж–ҮжЎЈжөҒиҝӣиЎҢе®ҡдҪҚгҖӮйҖҡиҝҮ вҖңleftвҖқгҖҒвҖқtopвҖқгҖҒвҖқrightвҖқ д»ҘеҸҠ вҖңbottomвҖқ еұһжҖ§жқҘи®ҫзҪ®жӯӨе…ғзҙ зӣёеҜ№дәҺиҮӘиә«дҪҚзҪ®зҡ„еҒҸз§»гҖӮ
  • stickyпјҡзІҳжҖ§е®ҡдҪҚгҖӮзү№жҖ§иҝ‘дјјдәҺrelativeе’Ңfixedзҡ„еҗҲдҪ“пјҢеғҸIOSйҖҡи®ҜеҪ•зҡ„вҖңйЎ¶еұҒиӮЎж•ҲжһңвҖқ

жіЁж„Ҹпјҡжө®еҠЁгҖҒз»қеҜ№е®ҡдҪҚе’Ңеӣәе®ҡе®ҡдҪҚдјҡи„ұзҰ»ж–ҮжЎЈжөҒпјҢзӣёеҜ№е®ҡдҪҚдёҚдјҡи„ұзҰ»ж–ҮжЎЈжөҒпјҢз»қеҜ№е®ҡдҪҚзӣёеҜ№дәҺиҜҘе…ғзҙ жңҖиҝ‘зҡ„е·Іе®ҡдҪҚзҡ„зҘ–е…Ҳе…ғзҙ пјҢеҰӮжһңжІЎжңүдёҖдёӘзҘ–е…Ҳе…ғзҙ и®ҫзҪ®е®ҡдҪҚпјҢйӮЈд№ҲеҸӮз…§зү©жҳҜbodyеұӮгҖӮ

  • йЎ№зӣ®пјҡ

    • flex-shrink зј©е°ҸжҜ”дҫӢпјҢй»ҳи®Өдёә 1гҖӮ

    • flex з®ҖеҶҷgrow shrink basis

    • flex: auto (1 1 auto) -> (ж”ҫеӨ§жҜ”дҫӢ1пјҢзј©е°ҸжҜ”дҫӢ1пјҢеҲҶй…ҚеӨҡдҪҷз©әй—ҙд№ӢеүҚеҚ жҚ®зҡ„дё»иҪҙз©әй—ҙ)
